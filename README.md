# 星翼突击

一款纯前端 Canvas 飞行战斗小游戏。

## 本地游玩

直接用浏览器打开 `index.html` 即可。

## 部署到公网

推荐使用 GitHub Pages：

1. 在 GitHub 创建一个新仓库。
2. 将本项目推送到该仓库的 `master` 或 `main` 分支。
3. 打开仓库的 `Settings -> Pages`。
4. 在 `Build and deployment` 中选择 `GitHub Actions`。
5. 等待 `Actions` 里的 `Deploy static game to GitHub Pages` 运行完成。

发布完成后，页面地址通常是：

```text
https://你的用户名.github.io/仓库名/
```

如果仓库名是 `你的用户名.github.io`，页面地址通常是：

```text
https://你的用户名.github.io/
```
